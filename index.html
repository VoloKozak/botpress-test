Bot testing

For testing please follow next steps:

  1. Import bot to your botpress server
  2. Upload assets to your botpress server
    • Open your bot from Bots page
    • Click on Code Editor and change it to Advanced
    • Click on 'Upload' button, choose your mcdonalds_assistant.css file and specify the following path: assets/modules/channel-web/mcdonalds_assistant.css
    • Click on 'Upload' button, choose your friesboxloader-cropped.gif file and specify the following path: assets/modules/channel-web/images/friesboxloader-cropped.gif
  3. Run the botpress server locally / Make sure you have a botpress server deployed
  4. Enter your host url in the input below and click 'Init bot'
